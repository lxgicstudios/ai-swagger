# OpenAPI/Swagger documentation generation utilities

OpenAPI/Swagger generator. Creates API documentation with interactive documentation.
## Usage

```javascript
// Import the package
const swagger = require('ai-swagger');

// Basic usage example
swagger.main();
```

## Examples

See the `examples/` directory for more detailed usage examples.

## Installation

```bash
npm install ai-swagger
```
